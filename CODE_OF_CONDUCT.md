# 행동강령

이 저장소의 issue, pull request, review, discussion, release 관련 대화와 연결된 오프라인 행사는
서로 존중하는 협업 공간이어야 합니다.

## 기대하는 행동

- 기술적 근거와 재현 가능한 정보로 의견을 제시합니다.
- 경험 수준, 정체성, 국적, 언어, 장애, 성별, 종교, 정치적 견해와 무관하게 존중합니다.
- review에서는 사람 대신 변경 사항과 위험을 비판합니다.
- security·privacy·license 우려를 조기에 제기하고 secret과 개인 데이터를 보호합니다.

## 허용하지 않는 행동

- 괴롭힘, 차별, 위협, 모욕, 원치 않는 성적 언어·이미지
- doxxing, credential·prompt·개인정보·private log 공개
- 취약점의 무단 공개, 서비스 방해, 악성 payload 공유
- 기여자·운영자를 사칭하거나 review·투표·issue를 조작하는 행위

## 신고와 조치

신고에는 필요한 최소 사실만 포함하고 개인정보·secret을 public issue에 올리지 않습니다. 민감한 신고는
[보안 정책](SECURITY.md)의 비공개 연락 절차를 사용해 repository maintainer에게 전달합니다. 일반 행동
문제도 public thread를 확대하기보다 비공개 연락 경로를 먼저 요청할 수 있습니다.

maintainer는 사실 확인, 경고, comment 수정·삭제, issue/PR 제한, 일시 또는 영구 참여 제한 등 상황에
맞는 조치를 취할 수 있습니다. 보복은 허용하지 않습니다.
